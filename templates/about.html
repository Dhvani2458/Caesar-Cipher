About This Project

This project demonstrates the Caesar Cipher, one of the oldest encryption techniques used in classical cryptography.


🧠 Purpose

To understand how substitution ciphers work and how basic encryption techniques can be implemented using Python and Flask.


⚙️ Technologies Used

  • Python (Flask Backend)
  • HTML, CSS, JavaScript
  • JSON for history storage

🚀 Features

  • Encrypt / Decrypt messages
  • Live web interface
  • History tracking system
  • Multi-page navigation
